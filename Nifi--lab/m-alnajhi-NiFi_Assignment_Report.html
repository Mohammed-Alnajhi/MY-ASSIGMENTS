📁 simulator.py (جزء من الكود)

def generate_record(seq_id):
    error_type = random.choice(["clean", "clean", "clean", "clean", "missing", "wrong_format"])
    
    if error_type == "missing":
        return {"id": seq_id, "amount": None, ...}
    elif error_type == "wrong_format":
        return {"id": str(seq_id) + "ERR", "amount": "invalid_amount", ...}
    else:
        return {"id": seq_id, "amount": random.uniform(10, 5000), ...}

📌 4. التحويلات (Transformations)

| العملية | Processor المستخدم | الإعدادات | |---------|-------------------|-----------| | استخراج الحقول | EvaluateJsonPath | `id`, `amount`, `status`, إلخ | | إضافة بيانات وصفية | UpdateAttribute | `source_type`, `processed_at` | | التصفية | RouteOnAttribute | `valid_data`, `missing_amount`, `invalid_amount` | | إزالة التكرار | DetectDuplicate | Identifier: `${filename}` | | تسجيل الفاسد | LogAttribute | ERROR level |

🎯 شروط RouteOnAttribute

valid_data      = ${amount:notNull():and(${amount:matches('\\d+(\\.\\d+)?')})}
missing_amount  = ${amount:isNull()}
invalid_amount  = ${amount:equals('invalid_amount')}
