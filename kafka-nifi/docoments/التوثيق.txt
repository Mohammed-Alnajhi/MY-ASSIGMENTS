توثيق مشروع Data Pipeline باستخدام Apache NiFi و Kafka

التاريخ: 17 مايو 2026

اسم المشروع: Real-Time Data Pipeline Using Apache NiFi and Kafka

1. ملخص المشروع
تم بناء خط أنابيب بيانات (Data Pipeline) متكامل يقوم بالآتي:

قراءة ملفات CSV من مجلد الإدخال

تقسيم الملفات الكبيرة إلى أجزاء بحجم 64 كيلوبايت

تحويل البيانات من صيغة CSV إلى صيغة JSON

إرسال البيانات إلى Kafka

استهلاك البيانات من Kafka

حفظ البيانات النهائية في مجلد الإخراج بصيغة JSON

5. تدفق البيانات (Data Flow)
5.1 مسار البيانات
text
[CSV Files] → [GetFile] → [SplitText] → [ConvertRecord] → [PublishKafka] → [ConsumeKafka] → [PutFile] → [JSON Output]
5.2 شرح كل مرحلة
المرحلة	الـ Processor	الوظيفة
1	GetFile	قراءة ملفات CSV من مجلد الإدخال
2	SplitText	تقسيم الملفات الكبيرة إلى أجزاء 64KB
3	ConvertRecord	تحويل CSV إلى JSON
4	PublishKafka	إرسال البيانات إلى Kafka Topic
5	ConsumeKafka	استهلاك البيانات من Kafka Topic
6	PutFile	حفظ البيانات في مجلد الإخراج
