⚙️ Setup Commands

Create Directories

cd C:\bigdata--lab
mkdir nifi_input nifi_output hadoop_data\namenode hadoop_data\datanode

Start Docker Services

docker compose up -d

Create Kafka Topics

docker exec kafka1 kafka-topics --create --bootstrap-server kafka1:19092 --topic nifi.orders.json --partitions 3 --replication-factor 1

Create HDFS Directory

docker exec hadoop-namenode hdfs dfs -mkdir -p /data
docker exec hadoop-namenode hdfs dfs -chmod 777 /data

Access Services

NiFi UI: http://localhost:8080/nifi (admin / admin123)
Kafka UI: http://localhost:8081
HDFS UI: http://localhost:9870

📈 Data Flow Summary

┌─────────────────────────────────────────────────────────────────────────────────────┐
│                                    DATA FLOW                                        │
├─────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                      │
│  Step 1:  Get CSV files from input directory                                        │
│           ↓                                                                          │
│  Step 2:  ListFile reads file names                                                 │
│           ↓                                                                          │
│  Step 3:  FetchFile retrieves file content                                          │
│           ↓                                                                          │
│  Step 4:  SplitContent splits into 64KB fragments                                   │
│           ↓                                                                          │
│  Step 5:  RouteText routes data based on content                                    │
│           ↓                                                                          │
│  Step 6:  UpdateAttribute adds metadata attributes                                  │
│           ↓                                                                          │
│  Step 7:  ConvertRecord converts CSV to JSON (if needed)                            │
│           ↓                                                                          │
│  Step 8:  PublishKafka sends JSON to Kafka topic                                    │
│           ↓                                                                          │
│  Step 9:  ConsumeKafka reads from Kafka topic                                       │
│           ↓                                                                          │
│  Step 10: PutHDFS writes JSON to HDFS with date partitioning                        │
│                                                                                      │
└─────────────────────────────────────────────────────────────────────────────────────┘
